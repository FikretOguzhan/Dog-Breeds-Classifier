# Dog-Breeds-Classifier
In this study, it is aimed to write an algorithm that can predict the breed of 120 different breeds of dogs using image processing. In addition, the test and validation accuracy results of different models using the same hyperparameters were compared and supported by graphics. Models: Xception, ResNet18, ResNet50, GoogleNet.
